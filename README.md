## Garden simulator
## About project
VR Garden Simulator is an early alpha XR/VR project developed in Godot using OpenXR tools.  
The project simulates a gardening experience in virtual reality where the player can interact with objects(e.g. planting flowers, watering them and collect coins) and explore a small garden environment.

## Idea and Goal
The idea of the project is to create a relaxing VR gardening simulation.  
The goal of this early alpha version is to demonstrate:
- VR environment setup  
- Player interaction in VR  
- Basic gardening mechanics  
- Core gameplay concept
